# Advanced Book Recommendation Engine
Sistema inteligente de recomendação de livros que utiliza múltiplos algoritmos de machine learning para fornecer sugestões personalizadas e precisas.

## Funcionalidades
- **Recomendação Baseada em Conteúdo**: Similaridade textual usando TF-IDF
- **Filtro Colaborativo**: Baseado em preferências de usuários
- **Sistema Híbrido**: Combina múltiplas abordagens para maior precisão
- **Análise de Sentimento**: Considera tom emocional dos livros
- **Recomendações Populares**: Ranking bayesiano de popularidade

## Tecnologias
- Python 3.8+
- TextBlob para NLP
- Type Hints para melhor manutenção
- Arquitetura modular e extensível
